Ennovar

March 2015 - March 2016 | Wichita, KS | Ennovar

Software Developer (Automation)

As a junion, I joined a team responsible for the development of an automated method of applying a storage array configuration, which has many configurable options and settings to a device. There were many different tools NetApp used to accomplish the task of applying a configuration to a storage array, but there was no easy way to move the configuration from a workstation to a storage array. An engineer would have to spend many hours manually entering the configuration settings in order to successfully place the configuration onto the array. This process wasted many engineering man-hours, and could be automated. The goal of this project was to automate the process of reading the configuration data from a database and applying it to an array.

This position required strong programming skills, especially in Java; an excellent knowledge of how to implement algorithms efficiently in code; the ability to thoroughly test and validate code; the ability to analyze and translate code from one language to another; and the ability to write clean, understandable code that would allow other developers to easily understand what the code was doing.
